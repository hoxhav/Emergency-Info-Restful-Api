FAQ About Emergency Service Directory

Note: This is a tool created for school purpose project.

How to use it?

This is an ESD developed by Vjori Hoxha. There is a form below which takes as input different fields depending on user wish. After that it populates a table with the required information. You can click the Name of the organization (also underlined to indicate that is clickable) and it will pop a dialog with more information

Can I use it multiple times?

Yes, but it will be okay if you would refresh the page once in a while ;)

Technologies used to develop this tool?

HTML5, CSS3, Bootstrap4, Javascript ES6, and JQuery
Search Criteria

Organization Name *can be partial - "vol" for "Volunteer"
